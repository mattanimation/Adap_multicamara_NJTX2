# Contenido del directorio

En este directorio se pueden encontrar los diseños de los siguientes módulos:

- [Módulo de cámara para el sensor OV5680](https://github.com/tidus747/Adap_multicamara_NJTX2/tree/master/Desarrollo_Electronico/Modulo_Camara) - Diseño para el módulo de cámara con sensor OV5680.
- [Módulo de cámara para el sensor OV5680 con pistas grandes](https://github.com/tidus747/Adap_multicamara_NJTX2/tree/master/Desarrollo_Electronico/Modulo_Camara_pistas_grandes) - Modificación con pistas más anchas para el módulo de cámara con sensor OV5680.
- [Módulo de depuración](https://github.com/tidus747/Adap_multicamara_NJTX2/tree/master/Desarrollo_Electronico/Modulo_Depuracion) - Diseño para el módulo de depuración de líneas de la Nvidia Jetson TX2.
- [Módulo para pruebas de CNC](https://github.com/tidus747/Adap_multicamara_NJTX2/tree/master/Desarrollo_Electronico/Modulo_pruebas_CNC) - Diseño para el módulo de pruebas de CNC.
- [Módulo para pruebas de CNC versión Tiny](https://github.com/tidus747/Adap_multicamara_NJTX2/tree/master/Desarrollo_Electronico/Modulo_pruebas_CNC_Tiny) - Diseño reducido para el módulo de pruebas de CNC.
- [Módulo puente para módulo de cámara basado en el sensor OV5680](https://github.com/tidus747/Adap_multicamara_NJTX2/tree/master/Desarrollo_Electronico/Modulo_puente) -  Diseño del módulo puente para el módulo de cámara basado en el sensor OV5680.
- [Módulo puente para módulo de cámara basado en el sensor OV5680 con VIAS MOD](https://github.com/tidus747/Adap_multicamara_NJTX2/tree/master/Desarrollo_Electronico/Modulo_puente_VIAS_MOD) - Diseño del módulo puente para el módulo de cámara basado en el sensor OV5680 con VIAS más grandes.
- [Módulo puente para módulo de cámara basado en el sensor OV5647/IMX219](https://github.com/tidus747/Adap_multicamara_NJTX2/tree/master/Desarrollo_Electronico/Modulo_puente_Raspberry_Camera) - Diseño para el módulo puente para los módulos de cámara con sensor OV5647/IMX219.
- [Módulo puente para módulo de cámara basado en el sensor OV5647/IMX219 sin EEPROM](https://github.com/tidus747/Adap_multicamara_NJTX2/tree/master/Desarrollo_Electronico/Modulo_puente_Raspberry_Camera_Dupont_NO_EEPROM) - Diseño para el módulo puente para los módulos de cámara con sensor OV5647/IMX219 sin EEPROM.
- [Diseño para el soporte de la Nvidia Jetson TX2](https://github.com/tidus747/Adap_multicamara_NJTX2/tree/master/Desarrollo_Electronico/Soporte_Jetson) - Diseño para el soporte de la Nvidia Jetson TX2.
